# Visão e Estratégia <Badge text="v2026" variant="note" />

> **"Nos tornarmos a maior plataforma de Operação Comercial Autônoma do Brasil."**

Não estamos construindo mais um CRM. Estamos criando uma **nova categoria de software**: o **Sistema Comercial Autônomo** — uma plataforma onde negócios inteiros rodam sua operação comercial sem intervenção humana, usando agentes de IA que executam, aprendem e evoluem sozinhos.

---

## Missão

Empoderar pequenas e médias empresas brasileiras a competir com grandes corporações, dando a elas acesso a operação comercial inteligente e autônoma — algo que antes exigia equipes enormes e orçamentos milionários.

---

## O que o TomikCRM não é

| O que parece ser | O que realmente é |
|---|---|
| Um CRM tradicional | Um **Sistema Comercial Autônomo** com CRM embutido |
| Uma ferramenta de automação | Uma **plataforma de agentes de IA** que trabalham por você |
| Um chatbot builder | Um **orquestrador de operação comercial** multicanal |
| Um dashboard de métricas | Um **sistema de observabilidade** com ação automática |

## Posicionamento Competitivo

### TomikCRM vs CRMs tradicionais

| Aspecto | Pipedrive / HubSpot | TomikCRM |
|---|---|---|
| Filosofia | Ferramenta para humanos | Sistema autônomo com agentes IA |
| Follow-up | Lembrete para o humano fazer | Agente executa automaticamente |
| Qualificação | Regras estáticas | IA dinâmica com evolução (AAES) |
| Canais | E-mail centric | WhatsApp-first + multicanal |
| Agendamento | Link de agendamento (humano) | Agente negocia e agenda via chat |
| Preço | US$49–99/user/mês | R$97–297/mês (BRL, nativo BR) |
| Evolução | Manual (usuário otimiza) | Automática (AAES otimiza) |

### TomikCRM vs Ferramentas de automação

| Aspecto | N8N / Make / Zapier | TomikCRM |
|---|---|---|
| Foco | Automação genérica | Operação comercial específica |
| Setup | Técnico (nodes, flows) | Wizard guiado, sem código |
| IA | Integração manual com OpenAI | Agentes nativos com contexto CRM |
| Observabilidade | Logs de execução | KPIs de negócio + métricas de agente |
| Evolução | Estática (fluxo fixo) | Dinâmica (AAES evolui sozinho) |
| CRM embutido | Não (precisa integrar) | Sim (Kanban + pipeline nativo) |

<Aside type="caution" title="Vantagem de pioneiro">
Não existe concorrente direto no Brasil que combine CRM + agentes IA autônomos + evolução automática + multicanal WhatsApp-first. O TomikCRM está criando a categoria **Operação Comercial Autônoma** antes que outros percebam que ela existe.
</Aside>

## Glossário

| Termo | Definição |
|---|---|
| **Agent** | Entidade de IA que executa tarefas comerciais autonomamente |
| **Agent Runtime** | Engine que executa pipelines de agentes de IA |
| **AAES** | Auto Agent Evolution System — evolui agentes automaticamente |
| **BYO** | Bring Your Own — cliente traz sua própria instância Supabase |
| **Crew** | Grupo de agentes que trabalham juntos em tarefa complexa (CrewAI) |
| **Follow-up** | Sequência automatizada de mensagens para nutrição de leads |
| **Gateway** | Integração com plataforma de pagamento (Asaas, Mercado Pago...) |
| **Handoff** | Transferência de conversa de agente IA para humano (LiveChat) |
| **Lead** | Contato potencial capturado via webhook, formulário ou manual |
| **MCP** | Model Context Protocol — permite agentes acessarem ferramentas externas |
| **MRR** | Monthly Recurring Revenue — receita recorrente mensal |
| **Multi-tenant** | Múltiplas organizações compartilham infraestrutura com isolamento |
| **Pipeline** | Funil de vendas Kanban OU sequência de execução de agente IA |
| **RAG** | Retrieval-Augmented Generation — busca contexto antes de gerar resposta |
| **RLS** | Row Level Security — isolamento de dados por organização no PostgreSQL |
| **Scoring** | Pontuação automática de leads por probabilidade de conversão |
| **Seat** | Licença de usuário — cada pessoa que acessa ocupa um seat |
| **Template (Meta)** | Mensagem pré-aprovada pela Meta para WhatsApp fora da janela de 24h |
| **tomik_hosted** | Modo onde o TomikCRM gerencia o Supabase do cliente |
| **Tool (MCP)** | Função que um agente pode chamar para executar ações externas |
| **Webhook** | Endpoint HTTP que recebe eventos externos e dispara ações |
